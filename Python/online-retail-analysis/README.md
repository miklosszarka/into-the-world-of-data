# Online Retail Data Analysis

## Project Overview

This project analyzes transactional data from an online retail business to understand revenue patterns, customer behavior, product performance, and geographic distribution.

The analysis is performed using Python and focuses on extracting business insights from raw data.

---

## Key Questions

- What drives revenue?
- Which products generate the most sales?
- How do customers behave?
- How strong is customer retention?
- How is revenue distributed across countries?

---

## Analysis Performed

- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Revenue and seasonality analysis  
- Product analysis  
- Customer analysis  
- RFM segmentation  
- Cohort (retention) analysis  

---

## Tools Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## Project Structure

- `online_retail_analysis.ipynb` → full analysis notebook
- `online_retail_II.csv` → dataset not included due to file size, download from here: https://archive.ics.uci.edu/ml/datasets/Online+Retail+II
- `README.md` → project description  

---

## Key Insights

- Revenue is concentrated in a small number of customers and products  
- Customer retention is relatively weak  
- A few products drive a large share of total sales  
- Revenue is geographically concentrated
