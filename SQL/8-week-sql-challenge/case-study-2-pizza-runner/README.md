# Case Study 2 – Pizza Runner

## Project Overview

This case study is part of the **8 Week SQL Challenge** created by **Danny Ma**.

The goal of this project is to analyze customer orders, runner performance, and delivery data using SQL.

---

## Business Problem

Pizza Runner wants to better understand how their business is performing by analyzing customer orders, delivery efficiency, and operational trends.

This analysis focuses on questions such as:
- How many pizzas were ordered and delivered?
- Which pizzas are the most popular?
- How successful are deliveries?
- How do customer preferences and delivery performance affect the business?

---

## Data Cleaning

Before solving the business questions, part of the dataset needed cleaning and transformation.

This included handling:
- null values
- inconsistent text formatting
- missing delivery information
- data preparation for analysis

---

This case study includes practice with:

- `JOIN`

- `CASE WHEN`
- `CTEs`

## Business Value

This analysis helps identify:

- order volume and customer demand
- delivery success and runner performance
- product popularity
- operational patterns that may affect customer satisfaction
