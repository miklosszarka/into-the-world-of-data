# 8 Week SQL Challenge

## Project Overview

This repository contains my solutions for the **8 Week SQL Challenge** created by **Danny Ma**.

The challenge is designed to develop practical SQL skills through realistic business case studies.  
Each case study focuses on a different type of business problem and helps strengthen analytical thinking, SQL querying, and data storytelling.

---

## Goals of This Project

- Practice SQL on real-world style datasets
- Improve problem-solving and analytical thinking
- Build a strong SQL portfolio for data analyst / BI roles
- Translate raw SQL work into business-focused insights

---

## SQL Skills Covered

Across the 8 case studies, I practice and improve skills such as:

- `SELECT` / `WHERE`
- `ORDER BY` / `LIMIT`
- `GROUP BY` / `HAVING`
- `JOINs`
- `CASE WHEN`
- `CTEs`
- `Subqueries`
- `Window Functions`
- `Aggregations`
- Business-oriented data analysis

---

## Structure

Each folder contains:

- a `solution.sql` file with my SQL queries
- a `README.md` file summarizing the case study and the main business questions
